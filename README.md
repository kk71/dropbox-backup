dropbox_backup
==============

备份文件或目录至dropbox

### 环境要求

* bash
* git
* [dropbox_uploader](https://github.com/andreafabrizi/Dropbox-Uploader)(能自动下载)

### 使用

在~/.dropbox_backup中保存你需要备份的目录或文件绝对路径。

```bash
    # cd
    # git clone https://github.com/kk71/dropbox_backup.git
    # cd dropbox_backup
    # chmod +x backup.sh
    # ./backup.sh
    # vim ~/.dropbox_backup
```
